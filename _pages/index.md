# Docit

Documentation static site generator. Build Markdown into a static site and deploy to `docs/`—perfect for GitHub Pages.

[Get Started](docs/installation) · [View on GitHub](https://github.com/ChrisThompsonTLDR/laravel-docit)

## Features

- **[Installation](docs/installation)** – Composer or monorepo. Add the `docit` script and run `composer docit`.

- **[Quick Start](docs/quick-start)** – Build, output to `docs/`, and deploy. Content lives in `_docs` and `_pages`.

- **[GitHub Actions](docs/github-actions)** – Reusable workflow for build-and-deploy on push.

- **[Customization](docs/customization)** – Site name, sidebar, navigation, and more via Hyde config.
